❤ 10 cose che amo del CSS

1. BACKGROUND-SIZE COVER

background-size: cover;

Che bello essere in homepage a tutto schermo!

BACKGROUND-SIZE COVER


			.cover.bassotti{
			  background: url(bassotto.jpg) center center no-repeat;
			  background-size: cover;
			  }

2. IMG RESPONSIVE

Due semplici istruzioni, e le immagini si adattano!


			img{
				max-width: 100%;
				height: auto;
				}

3. MEDIA QUERY

MEDIA QUERY: a ogni schermo le sue esigenze!


			@media (max-width: 500px;) and (orientation: portrait) {
				.mquery{
				 font-size: 1rem:
				 width: 100%;
				 }
				}

4. FLEXBOX Space Around

FLEXBOX Space Around: distribuire in maniera uniforme gli elementi!


div{
	display: flex;
	justify-content: space-around;
	}

5. FLEXBOX CENTER

con le istruzioni CENTER di Flexbox finalmente gestiamo con esattezza l'allineamento anche verticale


div{
	display: flex;
	justify-content: center;
	align-items: center;
	}

7. Grid

Amo il display:grid perchè mi permette finalmente di mettere i bassotti dove voglio...


				div{
					display: grid;
					grid-template-columns: repeat(8, 1fr);
					grid-template-rows: repeat(6, 5vw);
					grid-gap: 1.5rem; 
					}

8. Sass

Con il pre-processing di Sass creo CSS come in un linguaggio di programmazione


			/* VARIABILI */
			$coloreRosso: rgba(195, 8, 65, 0.5);
			$coloreGiallo: rgba(222, 239, 11, 0.5);

			div.o1{ background-color: $coloreRosso; }
			div.o2{ background-color: $coloreGiallo; }
